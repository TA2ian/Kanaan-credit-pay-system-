# دليل تثبيت وتشغيل مشروع مجموعة كنعان الذكية

هذا المشروع مبني باستخدام React (Vite) مع Express (Node.js) و Firebase.

## المتطلبات الأساسية
- Node.js (الإصدار 18 أو أحدث)
- npm (يأتي مع Node.js)

## خطوات التثبيت

### 1. تثبيت الحزم والمكتبات
يمكنك تشغيل أمر التثبيت القياسي:
```bash
# باستخدام npm (المقترح)
npm install

# أو باستخدام yarn (إذا كنت تفضله)
yarn install
```

إذا كنت تستخدم بنية Monorepo أو تفضل تشغيل الأمر في خطوة واحدة:
```bash
# مثال للمجلدات الفرعية (إذا تم نقل المشروع لاحقاً)
cd packages/app && yarn install
```

### 2. إعداد ملف البيئة
قم بإنشاء ملف `.env` في المجلد الرئيسي وأضف مفاتيح Firebase الخاصة بك (إذا لزم الأمر):
```env
# مثال على المتغيرات المطلوبة
GEMINI_API_KEY=your_gemini_key
```

## أوامر التشغيل

### تشغيل وضع التطوير
```bash
npm run dev
```

### بناء المشروع للإنتاج
```bash
npm run build
```

### تشغيل نسخة الإنتاج
```bash
npm run start
```

## التخلص من الملفات المؤقتة
```bash
npm run clean
```
